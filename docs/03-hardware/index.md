# S7-1200 硬件

SIMATIC S7-1200 是 SIMATIC S7 可编程控制器系列中的新型模块化微型 PLC，其组成为： 

- 控制器: 带有集成 PROFINET 接口，用于编程设备、HMI 或其它 SIMATIC 控制器之间通信
- 信号板: 可直接插入到控制器 
- 信号模块: 用于扩展控制器输入和输出通道 
- 通信模块: 用于扩展控制器通信接口 
- 附件: 如电源、开关模块、电池板或 SIMATIC 存储卡
