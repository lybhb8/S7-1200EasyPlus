## 常问问题

 **![](images/5.gif) 为何程序下载完毕后PLC无法运行，PLC诊断信息中出现“超出最大程序循环时间”？**  
  
可能的原因：

1、程序过长，执行时间超过设置的最大循环时间；

2、程序中有死循环程序导致执行时间超过最大循环时间；

3、定时中断程序的执行时间大于所设定的定时间隔时间，导致定时中断事件缓冲区溢出而停机。
