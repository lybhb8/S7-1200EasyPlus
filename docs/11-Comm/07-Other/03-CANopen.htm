CANopen 通信

PN/CAN LINK

图 1. 模块外观

订货号：6BK1620-0AA00-0AA0

S7-1200 通过该网关模块（Profinet 转 CAN）实现 CAN 通信，该模块可以作为一个 IO 设备与 S7-1200 进行通信，如图 2 所示。

图 2. 应用

功能概述：

  • PROFINET 与 CAN 总线 2.0A/B 或 CANopen 管理器或从站之间的数据交换（根据 CiA 301 和 302）
  • CANopen 特点：
    • 节点/寿命保护
    • 心跳
    • 同步（生产者/消费者）
    • 通过 TIA Portal V14 SP1 UPD2+HSP0197 或更高版本 TIA Portal 中组态
  • 集成 PROFINET 交换机和用于 CAN 的 D-sub 9 针
  • 最多 126 个 CAN 节点
  • 512 接收机和 512 发射机 PDO
  • 两个网络之间的电隔离
  • 诊断中断

功能/参数分配

SIMATIC PN/CAN 链路代表 PROFINET 和 CAN 或 CANopen（根据 CIA）之间的一个紧凑网关，为用户提供三种不同的操作模式：

  • 透明的
  • CANopen 管理者
  • CANopen 从站

CM CAN

图 3. 模块外观

该模块作为 ET200SP 分布式 IO 中的一个模块，该 ET200SP 站作为一个 IO 设备与 S7-1200 进行通信。

订货号：6ES7137-6EA00-0BA0

使用 CM CAN 通信模块，SIMATIC ET 200SP 现在可以直接与 CAN 和 CANopen 网络通信-最多 60 个 CAN 节点。该模块非常紧凑，宽度为 15mm，因此控制柜内的空间很小。

在 CANopen 模式下，可以导入或导出外部设备描述文件（.eds文件）。

具有三种工作模式的紧凑型通信模块：

  • 透明的
  • CANopen 管理者
  • CANopen 从站

通过 TIA Portal V15.1+HSP0310 或更高版本 TIA Portal 中组态。
