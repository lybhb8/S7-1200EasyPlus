硬件数据类型¶

硬件数据类型：硬件数据类型由 CPU 提供。 可用硬件数据类型的数目取决于 CPU。

根据硬件配置中设置的模块存储特定硬件数据类型的常量。 在用户程序中插入用于控制或激活已组态模块的指令时，可将这些可用常量用作参数。
