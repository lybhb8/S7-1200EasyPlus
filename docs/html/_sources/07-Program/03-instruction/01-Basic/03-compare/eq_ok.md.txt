## 参数的数据类型

|     |     |     |
| --- | --- | --- |
| ​参数 | ​数据类型 | ​说明 |

## 操作

|     |     |
| --- | --- |
| ​指令 | ​满足以下条件时 REAL 数测试结果为 TRUE： |
| ​OK | ​输入值为有效实数 [^1] |
| ​NOT_OK | ​输入值不是有效实数 [^1] |

[^1]: 如果 Real 或 LReal 类型的值为 +/- INF（无穷大）、NaN（不是数字）或者非标准化的值，则其无效。非标准化的值是非常接近于 0 的数字。CPU 在计算中用 0 替换非标准化的值。
